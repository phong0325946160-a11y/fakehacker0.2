# 🚀 FakeHacker v0.2 - Just For Fun

A purely spontaneous open-source project dedicated to everyone who loves the Windows Command Prompt (CMD). 
What started as a simple idea to "look cool" has now evolved into a fully interactive game!

## ✨ Key Features (15-in-1)
* Exclusive matrix-style percentage-loading effects.
* A dramatic 1-second maximum fan blast feature.
* And 13 other exciting features waiting for you to discover...
